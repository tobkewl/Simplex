# Simplex

This repository is the public home for **Simplex**.

Here you can:
- Download the latest version of the Simplex app
- Read release notes and changelogs
- Report bugs or request features for both the app and the website

---

## Download

Go to the **Releases** page and download the latest version.

If you are unsure which file to choose, download the most recent release.

---

## Installation notes

Simplex is currently distributed without code signing.

### Windows
You may see a “Windows protected your PC” warning.
Click **More info → Run anyway** to start the app.

---

## Feedback

You can report bugs or request features directly:
- From inside the Simplex app
- From the Simplex website

No extra accounts are required.

Please choose the correct type:
- **Bug report** if something is broken or not working as expected
- **Feature request** if you have an idea or improvement in mind

Clear descriptions help a lot when fixing or prioritizing things.
